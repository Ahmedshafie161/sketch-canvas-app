### 1. Enhanced Table Editing

#### Inline Editing (Like OneNote)
1. Create a table (Table tool â†’ drag to create)
2. Double-click any cell
3. Type directly - no popup!
4. Click outside or press Tab to confirm
5. Text formatting toolbar appears when editing

#### Cell Merging
1. Click first cell
2. Hold Shift + click more cells
3. Click "Merge" button in table toolbar
4. Cells combined with all text preserved

#### Cell Splitting
1. Click a merged cell
2. Click "Split" button
3. Cell divided back to original grid

#### Row/Column Management
1. Select any cell
2. Use "+ Row" to add row below
3. Use "+ Col" to add column to right
4. Use "- Row" or "- Col" to delete (requires selection)

#### Text Formatting
When editing a cell, you'll see:
- **Bold** button (B icon)
- *Italic* button (I icon)
- Underline button (U icon)
- Font size selector (8-72px)
- Text color picker
- Background highlight color picker

All changes apply in real-time!

### 2. Rich Media in Cells

#### Adding Images
1. Right-click cell
2. Select "Add Image / OCR Scan"
3. Choose image file
4. Image displayed with OCR text extracted automatically
5. Click image to view full-size

#### Adding GIFs
1. Right-click cell
2. Select "Add GIF"
3. Choose .gif file
4. Animated GIF plays in cell

#### Adding Videos
1. Right-click cell
2. Select "Add Video"
3. Choose video file
4. Video player with controls appears

#### Nested Tables
1. Right-click cell
2. Select "Add Nested Table"
3. Enter rows and columns
4. Complete sub-table in cell!

### 3. Voice Recording & Transcription

#### Recording Audio
1. Right-click any table cell
2. Click "Start Voice Recording"
3. Browser requests microphone permission (allow it)
4. Speak clearly and naturally
5. Watch live transcript appear
6. Click "Stop Recording" when done

#### What Gets Saved
- Audio file (playback in cell)
- Full transcript (editable text)
- Both stored in database
- Transcript searchable

#### Tips for Best Results
- Speak clearly and at normal pace
- Use in quiet environment
- Ensure good microphone quality
- Wait 1-2 seconds after starting before speaking

### 4. PDF Import with OCR

#### Importing PDFs
1. Click "Import" button in toolbar
2. Select PDF file
3. Wait for processing (progress in console)
4. Each page appears as:
   - Page image (left side)
   - Extracted text (right side, editable)

#### What You Can Do
- Highlight text with color tools
- Edit extracted text
- Add annotations
- Reorganize pages on canvas

#### OCR Quality
- Best with printed text
- Works with scanned documents
- Handwriting recognition (limited)
- Multiple languages supported

### 5. Animation System

#### Adding Animations
1. Select any object (shape, text, etc.)
2. Click "Animations" button in toolbar
3. Panel opens with options:
   - Fade In/Out
   - Slide
   - Rotate
   - Scale
   - Bounce
4. Set duration (100-5000ms)
5. Click "Add to Selected"

#### Playing Animations
- Click "Play All" to see all animations
- Animations play in order added
- Each plays based on its duration setting

#### Animation Types Explained
- **Fade**: Object fades out then in
- **Slide**: Object slides from right
- **Rotate**: Object spins 360Â°
- **Scale**: Object grows 1.5x then returns
- **Bounce**: Object jumps up and down

### 6. Drawing & Recognition

#### Pen/Stylus Drawing
1. Select "Draw" tool
2. Draw with mouse, finger, or stylus
3. Pressure sensitivity on supported devices
4. Smooth, responsive lines

#### Convert Drawing to Text (OCR)
1. Finish drawing
2. Right-click drawing
3. Select "Convert to Text (OCR)"
4. Handwriting recognized and converted
5. Edit text as needed

#### Convert to Shape
1. Right-click drawing
2. Select shape type:
   - Rectangle
   - Circle
   - Triangle
3. Drawing becomes clean geometric shape

### 7. Cloud Sync & Database

#### Local Storage (Automatic)
- All changes auto-saved to IndexedDB
- Works offline
- No setup required
- Data persists across sessions

#### Enable Cloud Sync
1. Click cloud icon in toolbar
2. Button shows "Synced" when enabled
3. All saves now push to cloud
4. Works across devices

#### Pull from Cloud
1. Enable cloud sync
2. Click "Pull" button
3. Latest data downloaded
4. Local database updated

#### Database Contents
- Users and authentication
- Folder hierarchy
- All files and canvases
- Canvas objects
- Connections and animations
- Voice recordings with transcripts
